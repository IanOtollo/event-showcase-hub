# Moses Musah Events | Premium Production Showcase

A high-end, bespoke digital platform for **Moses Musah Events**, a premier event production and management company based in Nairobi, Kenya.

## 🌟 Key Features

- **Immersive Experiences**: Scroll-triggered horizontal parallax animations for featured projects using Framer Motion.
- **Dedicated Service Verticals**: Robust, content-rich pages for Event Planning, Video Coverage, and Livestreaming.
- **Secure Management Portal**: 
    - Role-Based Access Control (RBAC) for Admins and Editors.
    - Professional Login system with Email/Password and integrated Google Authentication flow.
    - Dynamic Admin dashboard for media management with category-specific uploads.
- **Localized Design**: Fully rebranded for the Kenyan market with Nairobi-centric contact details and organization naming.
- **Advanced Navigation**: Seamless cross-page hash-link handling (`ScrollToHash`) for a fluid SPA experience.
- **Premium UI/UX**: Built with Radix UI primitives, Lucide icons, and a custom deep-blur navigation system.

## 🛠️ Tech Stack

- **Framework**: [React 18](https://reactjs.org/) with [TypeScript](https://www.typescriptlang.org/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **UI Components**: [Shadcn/UI](https://ui.shadcn.com/) (Radix UI)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State/Data**: [React Query](https://tanstack.com/query/latest)

## 🚀 Deployment

The project is optimized for deployment on **Vercel** or **Netlify**.

1. Connect your GitHub repository to Vercel.
2. Ensure the Build Command is `npm run build`.
3. Set the Output Directory to `dist`.
4. Deploy!
